# DESIGN.md

## 气质与意象

关键词：专业、可靠、极简。具象场景：一名运维工程师在凌晨的机房中，只点了一下鼠标，整套系统就自动部署完成，屏幕上只有清晰的进度条和绿色的勾。

## 视觉策略

- 图形语言：线性图标、无装饰
- 主色调：深蓝灰 + 绿色（成功状态）

## 配色方案

- 主色：#1a1a2e（深蓝灰，标题栏、进度框架）
- 成功色：#00c853（绿色，完成步骤指示器）
- 错误色：#ff1744（红色，失败步骤指示器）
- 进行中：#2196f3（蓝色，当前步骤指示器）
- 背景：#ffffff（白色）
- 文字：#333333（深灰）

## 字体排版

- 标题：系统默认粗体
- 正文：系统默认常规
- 日志：等宽字体（Consolas / Menlo）

## 页面结构

- 顶部：标题 + 副标题
- 中部：进度条 + 步骤列表
- 下部：滚动日志窗口
- 底部：操作按钮

## 设计禁忌

- 不使用渐变色
- 不使用圆角过大的卡片
- 不使用动画效果
- 不使用 emoji 作为 UI 元素
