安装说明

  1. 点击上方按钮下载对应系统的安装包
  2. Windows：右键 → 以管理员身份运行 OpenClaw_Installer.exe
  3. macOS：双击打开，如提示不安全请前往「系统设置 → 隐私与安全性」允许运行
  4. Linux：终端运行 chmod +x OpenClaw_Installer && sudo ./OpenClaw_Installer
  5. 点击「开始安装」，等待自动完成即可
