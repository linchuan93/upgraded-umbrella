核心特性

专为简化 OpenClaw 安装流程而设计

⚡

零环境依赖

无需预装 Python、Node.js 等任何运行环境，程序自带全部运行时，全新系统即可运行。

🤖

智能自动应答

三级降级策略自动处理 openclaw onboard 所有交互提示，无需用户手动输入。

🌍

国内镜像加速

自动检测网络环境，国内用户自动切换 npmmirror、清华 pip 镜像等国内源。

🔧

智能诊断修复

集成 openclaw doctor 诊断，安装失败时自动检测并修复常见问题。

🛡️

安全优先

默认 127.0.0.1 监听不暴露公网，关闭遥测，配置修改前自动备份。

🔄

自动重试机制

指数退避重试策略（最多 3 次），网络波动时自动恢复，无需重新启动。

项目架构

模块化设计，核心逻辑与 GUI 完全分离

# 入口层
main.py ─── GUI 模式 (tkinter) + CLI 降级

# 核心层
platform_detector   ─ 操作系统 / CPU / 磁盘 / 杀毒检测
dependency_manager  ─ Node.js / Python / Git 静默安装
network_manager     ─ 网络检测 + 国内镜像自动切换
openclaw_installer  ─ 官方脚本 / npm / brew 三级安装
onboard_handler     ─ non-interactive → PTY → 手动 三级应答
daemon_manager      ─ systemd / launchd / 任务计划 开机自启
doctor_runner       ─ 诊断 + 自动修复 + 错误码匹配
error_handler       ─ 指数退避重试 + 结构化错误报告
config_manager      ─ JSON 配置 + 自动备份 + 回滚

# 工具层
logger     ─ 统一日志 (控制台 + 文件轮转)
privilege  ─ UAC / sudo / pkexec 提权
security   ─ SHA-256 校验 + 安全配置验证
